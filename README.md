# Debt Tracking Project

## Özellikler
- Borç ekleme/kaldırma/düzenleme
- Borç türleri (Kredi kartı, kredi, fatura vb.)
- Ödeme tarihi takibi
- Görsel dashboard (D3.js ile)
- Borç özet tablosu
- CSV/Excel export

## Teknoloji Stacki
- Next.js 14
- Tailwind CSS
- D3.js
- Heroicons
- React Hook Form
- date-fns

## Kurulum
```bash
npm install
npm run dev
```
